# Selenium TypeScript Demo

Автотесты на `TypeScript` для проверки данных на Википедии с использованием `Selenium WebDriver`, `Mocha` и `Chai`.

## Требования

- Node.js 18+
- Google Chrome (актуальная версия)

## Установка

```bash
npm install
```

## Скрипты

- `npm test` — запускает все `.spec.ts` тесты через Mocha.
- `npm run build` — компилирует TypeScript (`tsc -p tsconfig.json`).

## Структура

- `src/tests/wikipedia.spec.ts` — два e2e-сценария:
  - Позитивный: открывает статью `Земля`, находит содержание кислорода (≈20.95%).
  - Негативный: открывает статью `Луна`, убеждается в отсутствии значения кислорода по тем же XPath.

## Настройка браузера

Мы используем пакет `chromedriver`, который подтягивает драйвер для текущего Chrome. Если корпоративный прокси блокирует загрузку — можно скачать `chromedriver` вручную и задать путь через переменную окружения `CHROMEDRIVER_PATH`.

## Полезные команды

```bash
npm test            # запуск тестов
npm run build       # компиляция в dist/
```
